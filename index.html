History Present Plans

I began my programming journey, with Python. I was mainly solving some simple problems, on my phone. It was not so entertaining to me at the moment, but it kept me busy between breaks in middle school.

Later in technical school, I learnt basics of C++ mixed with C. At first, I did not like it, but with time, I was favouring it more and more and soon C++ became my favorite programming language. With it, we analized some basic algorythms.

Other than that, I also got introduced to developing websited using HTML, CSS, JavaScript and later PHP with SQL. Learning all those stuff at the same time, was interesting and being able to create my own website, than adding some cool effects and finally making it more advanced by manipulating database, was very satisfying.

In collage, we dived into more details about C and C++, which was great and made me realize, how many things I was still missing. There was also introduction to algorythms and data-structures and computer graphics. Next I made a recap of computer networks and databases with SQL. Other classes introduced us to electornics and programming embedded systems, which I became very interested in.

Currently I am an applied computer science student at AGH University of Science and Technology. I also am an intern at Leading Devices, where I work mostly using Qt Qml in frontend to create desired graphical UI.

In my free time, I also program some simple programs, while trying to learn new languages, frameworks and generally improve my programming skills.

In the Future I am planning on learning more about embedded systems, helpful tools used to develop them and electronics together with commonly used interfaces and their standards.

Additionally, I would like to know how to use atleast one web framework, used to create web-applications etc. My goal would be learning atleast basics of, ruby on rails, NodeJS, TypeScript or React. Later on, I could move on to some less common frameworks.

But ofcourse, everything can change, depending on my future jobs.
